# g-autos

## Details
A Spring Boot CRUD API with persistence to PostgreSQL
- Java 11
- Spring Boot 2.7.6
- Gradle
- Archive output: Jar

## Requirements
- Java 11

## Usage
- Build
  ```
  ./gradlew build
  ```
  - Output directory: `build/libs`
- Run
  ```
  ./gradlew bootRun
  ```
- Test
  ```
  ./gradlew test
  ```
